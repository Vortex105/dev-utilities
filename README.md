# python_toolbox

A collection of small Python utilities and mini-projects useful for learning, automation, and quick tasks.

## Contents

- Utilities: scripts for common tasks (file manipulation, data parsing, small CLI tools).
- Mini-projects: self-contained examples and demos (games, algorithms, small web scrapers).
- Tests: simple unit tests or example usage for many tools.

## Project structure

- tools/          - reusable utilities and helper modules
- projects/       - mini-projects and demos (each in its own folder)
- examples/       - short scripts showing how to use utilities
- tests/          - basic tests and demo runs
- README.md       - this file

## Installation

1. Clone the repo:
	 git clone <repo-url>
2. Create a virtual environment and install dependencies if any:
	 python -m venv .venv
	 .venv\Scripts\activate        # Windows
	 source .venv/bin/activate     # macOS / Linux
	 pip install -r requirements.txt  # if file exists

## Usage

- Browse projects in projects/ and run their main script, e.g.:
	python projects/some_demo/main.py
- Import utilities in your own scripts:
	from tools.some_util import useful_function
